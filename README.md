# 日報システム
WEBブラウザで日報の作成・更新ができるシステムです。<br>
ユーザの管理からプロジェクトの作成も行えます。

## 使い方
https://www.fukuda-gikou.co.jp/performance/business/1655/

## 日報機能一覧
* ログイン
* ユーザ管理
    * プロフィール表示
    * プロフィール画像設定
    * 所属部署
    * 権限（一般、管理者）
    * ステータス（有効、無効）
* 日報
    * 出社日、出勤時刻、退勤時刻
    * プロジェクト毎の作業時間、進捗率、納期の入力
    * 作業内容の入力
* プロジェクト
    * ソート機能
* 部署
